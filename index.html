Basic knowledge about Analog and Digital Electronics

The basic logic gates are :

  1. AND GATE
  2. OR GATE
  3. NOT GATE

Using these logic gates we can design any complex circuit.

Some other GATES that are combinational logic circit and are grouped as Universal Gates :

  • NAND Gate : AND + NOT Gate
  • NOR Gate : OR + NOT Gate

Let us get some knowledge about these Logic Gates :

  1. AND GATE
    2. AND Gate

    This gate accepts minimum 2 inputs.

    This gate produces high voltage(O/P=1) only when all the inputs are high voltage(I/P=1).

    AND Gate is represneted by the following symbol :

    TRUTH TABLE (with two inputs):

    Input1 Input2 Output
    0 0 0 (low voltage)
    0 1 0 (low voltage)
    1 0 0 (low voltage)
    1 1 1 (high voltage)
  2. OR GATE
    3. OR Gate

    This gate accepts minimum 2 inputs.

    This gate produces high volate(O/P=1) when one or more input is high voltage(I/P=1).

    OR Gate is represneted by the following symbol :

    TRUTH TABLE (with two inputs):

    Input1 Input2 Output
    0 0 0 (low voltage)
    0 1 1 (high voltage)
    1 0 1 (high voltage)
    1 1 1 (high voltage)
  3. NOT GATE
    4. NOT Gate

    This gate accepts only one 1 input and negates the output.

    NOT gate produces low voltage (O/P=0) on recieving high voltage (I/P=1) and vice-versa.

    NOT Gate is represneted by the following symbol :

    TRUTH TABLE :

    Input1 Output
    0(low volate) 1 (high voltage)
    1(high voltage) 0 (low voltage)

*NOTE BELOW :

*High Volate = +5V which is represented by 1 in binary language.

*Low Volate = 0V which is represented by 0 in binary language.
